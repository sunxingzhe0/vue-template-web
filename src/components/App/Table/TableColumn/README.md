# table column custom 自定义表格显示字段

## Props

| name    | type       | descripation                 |
| :------ | :--------- | :--------------------------- |
| v-model | `object[]` | 表格字段详见[这里](#options) |

### `options`

| name     | type      | default | descripation       |
| :------- | :-------- | :------ | :----------------- |
| prop     | `string`  | -       | 字段名             |
| label    | `string`  | -       | 对应字段名的 label |
| checked  | `boolean` | -       | 是否选中           |
| disabled | `boolean` | `false` | 是否禁用           |
